# Сертификаты

Сюда нужно положить PDF-файлы с сертификатами.

## Структура папок:

```
documents/certificates/
├── sro-design.pdf          # Допуск СРО на проектирование
├── sro-build.pdf           # Допуск СРО на строительство
├── sro-recon.pdf           # Допуск СРО на реконструкцию
├── iso-9001.pdf            # ISO 9001 (качество)
├── iso-14001.pdf           # ISO 14001 (экология)
├── iso-45001.pdf           # ISO 45001 (охрана труда)
├── fire-safety.pdf         # Сертификат пожарной безопасности
├── fire-declaration.pdf    # Декларация пожарной безопасности
├── sentryglas.pdf          # Сертификат на Sentryglas
├── triplex.pdf             # Сертификат на триплекс
├── smart-glass.pdf         # Сертификат на смарт-стекло
├── gost.pdf                # Сертификат соответствия ГОСТ
└── license.pdf             # Лицензия на строительную деятельность
```

## Как добавить свой сертификат:

1. Положи PDF-файл в эту папку
2. Назови его по-английски, без пробелов (например: `my-cert.pdf`)
3. Открой файл `app/certificates/page.tsx`
4. Найди нужную категорию и добавь строку:
   ```js
   { name: 'Название сертификата', filename: 'имя-файла.pdf' },
   ```

## Пример:

Если у тебя есть сертификат "Допуск СРО на проектирование" и файл называется `sro-design.pdf`, то в категории `sro` будет:

```js
{ name: 'Допуск СРО на проектирование', filename: 'sro-design.pdf' },
```
