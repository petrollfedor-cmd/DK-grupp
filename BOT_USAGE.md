# Telegram Bot для управления сайтом

## Установка

1. Убедитесь, что токен бота и ID админа указаны в `.env.local`:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен
   TELEGRAM_ADMIN_IDS=ваш_id
   ```

2. Запустите бота:
   ```bash
   node bot.js
   ```

## Использование

1. Откройте Telegram и найдите вашего бота
2. Отправьте команду `/start`
3. Выберите раздел для редактирования:
   - 📋 Навигация
   - 🖼 Hero секция
   - 🏗 Проекты
   - 📎 Футер
   - 📊 Статистика

## Формат данных

### Навигация
```
Главная|/#hero
Проекты|/#projects
Типы работ|/types-works
```

### Проекты
```
Название проекта|Описание|/путь/к/фото
```

### Hero секция
- Заголовок: один текст
- Описание: один текст
- Фото: путь к изображению

## API Endpoints

- `GET /api/navigation` - данные навигации
- `GET /api/hero` - данные hero секции
- `GET /api/projects` - данные проектов
- `GET /api/footer` - данные футера
