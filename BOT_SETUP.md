# 🤖 Telegram Bot для управления контентом сайта

## Что сделано

Бот позволяет управлять всем контентом сайта через Telegram:
- **Проекты** — добавлять, редактировать, удалять, менять фото/описание
- **Типы работ** — управлять карточками на странице "Типы работ"
- **О нас** — редактировать текст страницы
- **Контакты** — менять телефон, email, адрес

## Установка

### 1. Создать бота в Telegram
1. Напишите @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. **Скопируйте токен** (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Получить свой Telegram ID
1. Напишите @userinfobot
2. Скопируйте ваш ID (например: `123456789`)

### 3. Настроить.env
```bash
cp .env.example .env.local
```

Откройте `.env.local` и заполните:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_ADMIN_IDS=ваш_telegram_id
```

### 4. Настроить webhook
```bash
# Получите ваш домен (для локалки используйте ngrok)
# Установите webhook:
curl -X POST "https://api.telegram.org/botYOUR_TOKEN/set_webhook?url=https://YOUR_DOMAIN/api/bot/webhook"

# Для локальной разработки с ngrok:
ngrok http 3000
# Затем:
curl -X POST "https://api.telegram.org/botYOUR_TOKEN/set_webhook?url=https://YOUR_ngrok_url/api/bot/webhook"
```

### 5. Запустить проект
```bash
yarn dev
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/help` | Список команд |
| `/projects` | Управление проектами |
| `/work-types` | Управление типами работ |
| `/about` | Редактирование "О нас" |
| `/contacts` | Редактирование контактов |

## Как редактировать контент

### Через кнопки:
1. `/start` → выберите раздел
2. Выберите элемент из списка
3. Нажмите на поле для редактирования
4. Отправьте новый текст в чат
5. Нажмите "✅ Подтвердить" или "❌ Отмена"

### Пример:
```
Вы: /start
Бот: 👋 Панель управления... [кнопки меню]
Вы: [нажимаете "📁 Проекты"]
Бот: 📁 Проекты: [список проектов с кнопками]
Вы: [нажимаете на проект]
Бот: 📌 Название проекта [кнопки редактирования]
Вы: [нажимаете "✏️ Название"]
Бот: ✏️ Название\nТекущее: Аэропорт Гагарин\nОтправьте новое:
Вы: Аэропорт Домодедо
Бот: ✅ "title" обновлено!
```

### Изменить фото:
1. Выберите проект → "🖼 Фото"
2. Отправьте фото как изображение в Telegram
3. Или отправьте URL изображения

## Структура данных

Все данные хранятся в JSON-файлах:
- `data/projects.json` — проекты на главной
- `data/work-types.json` — типы работ
- `data/about.json` — страница "О нас"
- `data/contacts.json` — контакты

## API Endpoints

- `GET /api/content/projects` — получить проекты
- `PUT /api/content/projects` — обновить проекты
- `POST /api/bot/webhook` — webhook для Telegram

## Безопасность

- Бот обрабатывает команды только от администраторов (указанных в `TELEGRAM_ADMIN_IDS`)
- Токен бота хранится в `.env.local` и не выкладывается в git
